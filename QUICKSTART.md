# üöÄ QUICK START GUIDE

## Heart Disease Prediction System - Django

### ‚ö° Fast Setup (5 minutes)

#### 1. Install Dependencies
```bash
cd heart_disease_project
pip install -r requirements.txt
```

#### 2. Setup Database
```bash
python manage.py makemigrations
python manage.py migrate
```

#### 3. Prepare Data
```bash
# Copy your dataset
cp /path/to/heart_disease_combined.csv data/

# Note: The file should be at data/heart_disease_combined.csv
```

#### 4. Train Models
```bash
python train_model.py
```
This creates model files in `models_data/` directory.

#### 5. Run Server
```bash
python manage.py runserver
```

#### 6. Open Browser
Visit: **http://127.0.0.1:8000**

## üîê Blockchain Implementation

### How It Works:

1. **User makes prediction** ‚Üí Saved to database
2. **Blockchain record created** ‚Üí Linked to prediction
3. **Results page** ‚Üí Shows prediction + brief blockchain summary
4. **Blockchain page** ‚Üí Shows complete chain with all hashes

### Blockchain Features:
- SHA-256 hashing
- Chain linking (each block references previous)
- Genesis block
- Immutable audit trail
- Separate page for inspection

## üõ†Ô∏è Troubleshooting

### Problem: "Model files not found"
**Solution**: Run `python train_model.py` first

### Problem: "Database error"
**Solution**: Run migrations:
```bash
python manage.py makemigrations
python manage.py migrate
```

### Problem: "Static files not loading"
**Solution**: 
```bash
python manage.py collectstatic
```

### Problem: "ModuleNotFoundError: No module named 'django'"
**Solution**: 
```bash
pip install -r requirements.txt
```
