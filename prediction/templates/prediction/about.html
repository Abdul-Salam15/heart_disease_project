About HeartCare AI

Advanced AI-Powered Heart Disease Prediction System

Overview

HeartCare AI is a cutting-edge machine learning system designed to predict the risk of heart disease based on patient clinical data. Our system combines multiple AI models with blockchain security to provide accurate, transparent, and secure medical predictions.

Key Features

Dual AI Models

We use both Logistic Regression and Random Forest algorithms to provide comprehensive risk assessment with consensus predictions.

Blockchain Security

Every prediction is logged to an immutable blockchain, ensuring data integrity, transparency, and complete audit trail.

Feature Importance

Understand which clinical factors contribute most to the prediction, helping clinicians make informed decisions.

High Accuracy

Our models achieve high accuracy with comprehensive evaluation metrics including precision, recall, and ROC-AUC scores.

How It Works

1. Data Input

Enter patient clinical measurements and symptoms

2. AI Analysis

Dual models analyze data and generate predictions

3. Risk Assessment

Receive consensus prediction with probability scores

4. Blockchain Log

Prediction securely recorded to blockchain

Technology Stack

Machine Learning
  • Logistic Regression (Scikit-learn)
  • Random Forest Classifier (200 trees)
  • Feature scaling with StandardScaler
  • Cross-validation for robust evaluation
Blockchain
  • SHA-256 cryptographic hashing
  • Immutable record chain
  • Database-backed persistence
  • Integrity verification system
Web Framework
  • Django 4.2 (Python)
  • Bootstrap 5 (Responsive UI)
  • SQLite Database
  • RESTful Architecture
Data Science
  • Pandas (Data processing)
  • NumPy (Numerical computing)
  • Matplotlib & Seaborn (Visualization)
  • Joblib (Model persistence)

Clinical Features Used

Our model analyzes the following patient parameters:

Numerical Features:
  • Age
  • Resting Blood Pressure
  • Cholesterol Level
  • Maximum Heart Rate
  • ST Depression (Oldpeak)
  • Fasting Blood Sugar
Categorical Features:
  • Sex
  • Chest Pain Type
  • Resting ECG Results
  • Exercise-Induced Angina
